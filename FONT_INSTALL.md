# Инструкция по установке шрифта Calson

## Текущая ситуация
CSS настроен для использования шрифта Calson, но файлы шрифта еще не добавлены в проект.

## Что нужно сделать:

### Вариант 1: Добавить файлы шрифта вручную

1. Получите файлы шрифта Calson (если у вас есть лицензия):
   - `Calson-Plain-Cyrillic.woff2` (или `.woff`, `.ttf`)
   - `Calson-Bold-Cyrillic.woff2` (или `.woff`, `.ttf`)

2. Скопируйте файлы в папку:
   ```
   /Users/alinagodm/Desktop/Transmarine/public/fonts/
   ```

3. Убедитесь, что имена файлов точно соответствуют тем, что указаны в `src/index.css`:
   - `Calson-Plain-Cyrillic.woff2`
   - `Calson-Bold-Cyrillic.woff2`

### Вариант 2: Экспорт из Figma

Если шрифт используется в вашем Figma файле:
1. Откройте Figma файл
2. Выберите текстовый элемент с шрифтом Calson
3. В правой панели нажмите на название шрифта
4. Если доступно, используйте плагин для экспорта шрифтов (например, "Font Export")

### Вариант 3: Использовать альтернативу

Пока файлы шрифта не добавлены, браузер будет использовать fallback шрифты:
- Arial (если Calson не найден)
- Times New Roman (если Arial не найден)

## После добавления файлов

После того как вы добавите файлы шрифта в `/public/fonts/`, страница автоматически начнет использовать Calson вместо fallback шрифтов.
