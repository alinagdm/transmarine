# Настройка API погоды

Компонент Weather использует бесплатный API open-meteo.com, который не требует API ключа и работает сразу.

## Текущая настройка

По умолчанию используется **open-meteo.com API** - бесплатный сервис без необходимости регистрации.

## Альтернатива: OpenWeatherMap API

Если вы хотите использовать OpenWeatherMap API (более детальные данные):

1. Зарегистрируйтесь на https://openweathermap.org/api
2. Получите бесплатный API ключ
3. Создайте файл `.env` в корне проекта
4. Добавьте строку:
   ```
   VITE_WEATHER_API_KEY=ваш_api_ключ_здесь
   ```
5. Перезапустите dev сервер

## Координаты

Текущие координаты: Калининград (54.7104, 20.4522)

Если нужно изменить город, отредактируйте константы `KALININGRAD_LAT` и `KALININGRAD_LON` в файле `src/components/Weather/Weather.tsx`
