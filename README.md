# TransMarine

Веб-сайт компании TransMarine - линейные и трамповые агенты / экспедиторы с 1996 года.

## Технологии

- React 18
- TypeScript
- Vite
- CSS3

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

## Структура проекта

```
src/
  components/
    Header/       # Шапка сайта с навигацией
    Hero/         # Главный баннер
    Services/     # Секция услуг со слайдером
    ServiceCard/  # Карточка услуги
    History/      # Секция истории компании
    Footer/       # Подвал сайта
    Arrow/        # Компонент стрелки
  App.tsx         # Главный компонент
  main.tsx        # Точка входа
```

## Особенности

- Адаптивный дизайн (desktop версия)
- Компонентная архитектура
- Типизация TypeScript
- Модульные CSS стили
